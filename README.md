# GoLucky 喜运达官网项目

## 项目概述
为喜运达（GoLucky）科技物流公司设计并开发全新国际化官网

## 设计风格
- 科技极简 + 全球化 + 物流动线感
- 色彩：#0066FF(主蓝) #00D4FF(霓虹青) #FF6B35(活力橙) #0A0F1C(深空黑)

## 技术栈
- Next.js 14 + React 18
- Tailwind CSS
- Framer Motion
- 部署：Vercel / Netlify

## 页面结构
1. Hero Section - 首屏视觉冲击
2. Services - 四大核心服务
3. Data Visualization - 实时数据大屏
4. Network Map - 网络覆盖地图
5. Why Us - 六大核心优势
6. Cases - 客户案例
7. Partners - 合作伙伴
8. CTA - 行动召唤
9. Footer

## 多语言支持
- 中文(默认) / English / ภาษาไทย / Tagalog / Bahasa Indonesia

## 交付要求
- 响应式设计(Desktop/Tablet/Mobile)
- 流畅动画效果
- SEO优化
- 可访问性支持
